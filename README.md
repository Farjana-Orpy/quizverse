# 🎯 QuizVerse

A full-stack MERN Quiz Platform built using React, Node.js, Express, and MongoDB.

Users can create accounts, log in securely, take quizzes, track scores, and compete on the leaderboard.

---

# 🚀 Features

- 🔐 JWT Authentication
- 👤 User Signup & Login
- 🧠 Dynamic Quiz System
- 📊 Leaderboard
- 🌙 Dark Mode
- 📱 Responsive Design
- 🛡 Protected Routes
- ☁ MongoDB Atlas Database

# ▶️ How to Run the Project

## Clone Repository

```bash
git clone https://github.com/Farjana-Orpy/quizverse.git
```

## Install Frontend

```bash
cd client
npm install
npm run dev
```

## Install Backend

```bash
cd server
npm install
node index.js
```

---

# 🔐 Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```
